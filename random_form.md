###随机生成form表单
#####类内方法
- method

  获取类内方法
  
- string
    
  ``生成随机字符串，参数为数组[字段名称]``
  
- getForm

  ```获取随机表单
  
  参数1 ：为数组[数据类型 => [字段名称]]
  参数2 : debug 开启则直接打印postman form-data 格式数据，否则输出标准json
  ```
- int

  ```
  生成随机数字
  
  参数1 ： 为数组同string
  
  参数2 : min最小值
  
  参数3 ： max 最大值
  ```
  
- english

  ```
   生成随机翻译后的的数据 ，该方法较为消耗内存，不建议过大批量使用 
   参数1 :数组[数据类型 => [字段名称]]  
  ```
  
- name

    ```
    生成随机姓名
    参数同string
    ```
  
- json

  ```
    生成json格式数据,并支持json嵌套和json翻译
    参数1 ：数组 [字段名称 => [数据类型 => [字段名称]]]  
  ```
  
- district

    ```
        生成随机完整地区以斜线分割（到县级）
        参数1 数组 [字段名称] 注：字段名称以_cn结尾，则连同id串同时输出
    ```

- category

  ```
    生成category字典项
    参数1 数组 [字段名称] 注：如果字段数组的key以cn结尾，则连带字典项ID串同时输出
  ```

此外包含浮点，手机号，邮箱，网站，文件名称等等数据生成，同string使用方法
